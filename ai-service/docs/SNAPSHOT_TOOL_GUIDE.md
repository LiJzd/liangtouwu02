# 猪场视频截图识别工具使用指南

## 功能概述

`capture_pig_farm_snapshot` 工具为猪BOT智能体提供了实时视频截图识别能力。当用户询问猪场当前状况时，BOT可以自动截取视频帧并使用YOLO模型进行目标检测，返回检测到的猪只数量、位置和状态信息。

## 工作原理

```
用户询问猪场状况
    ↓
BOT智能体调用 capture_pig_farm_snapshot 工具
    ↓
从视频文件中读取当前帧
    ↓
使用YOLO模型进行目标检测
    ↓
解析检测结果（类别、置信度、位置）
    ↓
返回结构化的JSON结果
    ↓
BOT将结果转换为自然语言回复用户
```

## 工具参数

### 输入参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| video_file | string | 否 | 自动查找 | 视频文件名，位于 `resources/assets` 目录 |
| confidence | int | 否 | 50 | 置信度阈值（0-100），越高越严格 |

### 输出结果

返回JSON格式的检测结果：

```json
{
  "success": true,
  "video_file": "pig_farm_video.mp4",
  "detection_count": 3,
  "detections": [
    {
      "class_name": "pig",
      "confidence": 0.856,
      "bbox": {
        "x1": 0.234,
        "y1": 0.456,
        "x2": 0.567,
        "y2": 0.789
      }
    }
  ],
  "summary": "检测到 3只pig"
}
```

## 使用场景

### 场景1: 用户询问猪场当前状况

**用户**: "帮我看看猪场现在的情况"

**BOT行为**:
1. 识别用户意图（查看猪场状况）
2. 调用 `capture_pig_farm_snapshot` 工具
3. 获取检测结果
4. 生成自然语言回复："我刚查看了猪场视频，检测到3只猪，它们分布在不同位置，状态良好。"

### 场景2: 用户询问猪只数量

**用户**: "现在猪场有多少只猪？"

**BOT行为**:
1. 调用 `capture_pig_farm_snapshot` 工具
2. 统计检测到的猪只数量
3. 回复："根据视频监控，当前检测到3只猪。"

### 场景3: 用户要求高精度检测

**用户**: "帮我仔细检查一下猪场，要准确的"

**BOT行为**:
1. 调用工具时设置更高的置信度阈值
2. `capture_pig_farm_snapshot("confidence=80")`
3. 返回更可靠的检测结果

## 技术细节

### 视频处理流程

1. **视频文件定位**: 
   - 默认在 `resources/assets` 目录查找视频文件
   - 支持 .mp4, .avi, .mov, .mkv 格式

2. **帧提取**:
   - 使用 OpenCV 读取视频
   - 提取当前帧（第一帧）
   - BGR 格式的 numpy 数组

3. **目标检测**:
   - 使用 YOLOv10 模型
   - 可配置置信度阈值
   - 返回类别、置信度、边界框

4. **结果解析**:
   - 坐标归一化（0-1范围）
   - 统计各类别数量
   - 生成友好的摘要信息

### 性能优化

- **模型单例**: YOLO模型只加载一次，后续调用复用
- **快速截帧**: 只读取第一帧，避免处理整个视频
- **异步处理**: 支持异步调用，不阻塞其他操作

### 错误处理

工具会处理以下错误情况：

1. **视频文件不存在**: 返回错误信息，提示检查文件路径
2. **视频无法打开**: 返回错误信息，可能是格式不支持
3. **无法读取帧**: 返回错误信息，可能是视频损坏
4. **模型加载失败**: 返回错误信息，检查模型文件

## 故障排查

### 问题1: 找不到视频文件

**症状**: 返回 "未找到可用的视频文件"

**解决方案**:
1. 检查 `resources/assets` 目录是否存在
2. 确认目录中有视频文件（.mp4, .avi, .mov, .mkv）
3. 检查文件权限

### 问题2: 无法打开视频

**症状**: 返回 "无法打开视频文件"

**解决方案**:
1. 确认视频格式是否支持
2. 尝试使用其他视频文件
3. 检查 OpenCV 是否正确安装

### 问题3: 检测不到目标

**症状**: detection_count 为 0

**解决方案**:
1. 降低置信度阈值（如 confidence=30）
2. 检查视频内容是否包含目标
3. 确认 YOLO 模型是否正确加载

### 问题4: 模型加载失败

**症状**: 返回 "AI 模型服务未就绪"

**解决方案**:
1. 检查模型文件路径是否正确
2. 确认 Ultralytics 库已安装
3. 查看日志获取详细错误信息
