# BOT视频截图识别功能集成说明

## 概述

本次更新为猪BOT智能体添加了视频截图识别功能，用户可以通过自然语言询问猪场当前状况，BOT会自动截取视频帧并使用YOLO模型进行识别，返回检测结果。

```python
async def tool_capture_pig_farm_snapshot(arg: str) -> str:
    # 从视频文件截取当前帧
    # 使用YOLO模型进行目标检测
    # 返回结构化的JSON结果
```

**功能特性**：
- 自动查找视频文件（支持 .mp4, .avi, .mov, .mkv）
- 支持自定义置信度阈值（默认50%）
- 返回检测到的目标数量、类别、置信度和位置
- 生成友好的摘要信息

## 工作流程

```
用户发送消息
    ↓
关键词匹配（_requires_tool）
    ↓
强制调用工具（force_tool=True）
    ↓
Agent选择 capture_pig_farm_snapshot 工具
    ↓
从视频文件读取当前帧
    ↓
YOLO模型进行目标检测
    ↓
解析检测结果（类别、置信度、位置）
    ↓
返回JSON格式结果
    ↓
Agent将结果转换为自然语言
    ↓
返回给用户
```

## 技术细节

### 视频处理

- **视频路径**: `resources/assets/` 目录
- **支持格式**: .mp4, .avi, .mov, .mkv
- **帧提取**: 使用 OpenCV 读取第一帧
- **图像格式**: BGR numpy 数组

### 目标检测

- **模型**: YOLOv10（复用感知控制器的模型单例）
- **置信度阈值**: 默认 0.5（可通过参数调整）
- **坐标归一化**: 边界框坐标归一化到 0-1 范围

### 返回格式

```json
{
  "success": true,
  "video_file": "pig_farm_video.mp4",
  "detection_count": 3,
  "detections": [
    {
      "class_name": "pig",
      "confidence": 0.856,
      "bbox": {
        "x1": 0.234,
        "y1": 0.456,
        "x2": 0.567,
        "y2": 0.789
      }
    }
  ],
  "summary": "检测到 3只pig"
}
```

## 故障排查

### 问题：BOT没有调用工具

**可能原因**：
1. 用户消息中没有触发关键词
2. Agent没有正确识别意图

**解决方案**：
1. 检查 `_requires_tool` 函数中的关键词列表
2. 查看日志中的 `[智能体]` 输出，确认是否触发工具调用
3. 尝试使用更明确的关键词（如"监控"、"视频"、"图片"）

### 问题：找不到视频文件

**可能原因**：
1. `resources/assets` 目录不存在
2. 目录中没有支持的视频文件

**解决方案**：
1. 确认目录路径正确
2. 添加视频文件到该目录
3. 检查文件扩展名是否支持

### 问题：检测不到目标

**可能原因**：
1. 置信度阈值过高
2. 视频内容中没有目标
3. YOLO模型未正确加载

**解决方案**：
1. 降低置信度阈值（如 confidence=30）
2. 检查视频内容
3. 查看感知控制器日志
