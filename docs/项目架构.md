## 项目概览

两头乌智慧养殖系统是一个三端分离的全栈项目：

```
┌─────────────────────────────────────────────────────┐
│                   两头乌智慧养殖系统                  │
└─────────────────────────────────────────────────────┘
           │                │                │
           ▼                ▼                ▼
    ┌──────────┐     ┌──────────┐     ┌──────────┐
    │   前端    │     │   后端    │     │ AI服务   │
    │  Vue 3   │────▶│  Spring  │────▶│ FastAPI  │
    │ TypeScript│     │   Boot   │     │ LangChain│
    └──────────┘     └──────────┘     └──────────┘
         │                │                │
         └────────────────┴────────────────┘
                         │
                         ▼
                  ┌──────────┐
                  │  MySQL   │
                  │ 数据库   │
                  └──────────┘
```

## 当前结构

### 根目录（混乱）

```
两头乌/
├── 两头乌后端/              ❌ 中文目录名
├── 两头乌前端/              ❌ 中文目录名
├── 两头乌ai端/              ❌ 中文目录名
├── 模型_视频_PPT/           ❌ 中文目录名
├── liangtouwu-common/       ⚠️ 重复的common
├── logs/                    ❌ 应该在各服务内
├── pip-cache/               ❌ 临时文件
├── pip-tmp/                 ❌ 临时文件
├── -p/                      ❌ 空目录
├── *.log                    ❌ 大量日志文件
├── *.txt                    ❌ 临时文本文件
└── *.bat                    ⚠️ 应该在scripts/
```

**问题**：
- 🔴 中文目录名不利于跨平台
- 🔴 临时文件和日志散落各处
- 🔴 缺少统一的文档目录
- 🔴 缺少统一的脚本目录



   - `ai-service/app/core/agent/tools.py` 添加工具定义
