# 两头乌项目结构说明

## 目录结构

```
两头乌/
├── backend/              # 后端服务 (Spring Boot)
├── frontend/             # 前端应用 (Vue 3)
├── ai-service/           # AI服务 (FastAPI + ReAct Agent)
├── resources/            # 共享资源
│   └── assets/          # 模型、视频、PPT等
├── database/            # 数据库脚本
│   ├── schema.sql      # 数据库结构
│   └── README.md       # 数据库说明
├── scripts/             # 启动和管理脚本
│   ├── start-all.bat   # 一键启动所有服务
│   └── ...
└── docs/                # 项目文档
```

## 快速启动

### 启动所有服务
```bash
scripts\start-all.bat
```

### 单独启动

后端:
```bash
cd backend
mvn spring-boot:run
```

前端:
```bash
cd frontend
npm run dev
```

AI服务:
```bash
cd ai-service
python main.py
```

## 服务地址

- 后端: http://localhost:8080
- 前端: http://localhost:5173
- AI服务: http://localhost:8000

## 主要功能

### 后端 (Spring Boot)
- 用户管理
- 猪场管理
- 数据API

### 前端 (Vue 3)
- 用户界面
- 数据可视化
- 实时监控

### AI服务 (FastAPI)
- ReAct智能体
- 猪场视频分析 (YOLO)
- QQ Bot集成
- RAG知识库
