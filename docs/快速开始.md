# 两头乌智慧养殖系统 - 快速开始指南

## 🚀 5分钟快速启动

### 第一步：环境检查
确保已安装以下工具：
- ✅ Node.js 18+ (`node --version`)
- ✅ Java 17+ (`java --version`)
- ✅ Python 3.10+ (`python --version`)
- ✅ MySQL 8.0+ (`mysql --version`)

### 第二步：数据库初始化
```bash
# 1. 启动 MySQL 服务
# Windows: 服务管理器启动 MySQL

# 2. 创建数据库
mysql -u root -p
CREATE DATABASE pig_farm CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
exit;

# 3. 导入表结构
mysql -u root -p pig_farm < database/schema.sql
```

### 第三步：启动所有服务

#### 方式一：一键启动（推荐）
```bash
scripts\start-all.bat
```

#### 方式二：分步启动

**1. 启动 AI 服务**
```bash
cd ai-service
pip install -r requirements.txt
python main.py
# 访问: http://localhost:8000/docs
```

**2. 启动后端服务**
```bash
cd backend
mvn clean install
mvn spring-boot:run -pl liangtouwu-app
# 访问: http://localhost:8080
```

**3. 启动前端服务**
```bash
cd frontend
npm install
npm run dev
# 访问: http://localhost:5173
```

## ⚡ 常见问题

### 1. 端口冲突
- **8000 端口被占用**: AI 服务无法启动
- **8080 端口被占用**: 后端服务无法启动
- **5173 端口被占用**: 前端服务无法启动

**解决方案**:
```bash
# Windows 查看端口占用
netstat -ano | findstr :8000

# 终止占用进程
taskkill /PID [进程ID] /F
```

### 2. 依赖安装失败
```bash
# 前端依赖
cd frontend
rm -rf node_modules package-lock.json
npm cache clean --force
npm install

# Python 依赖
cd ai-service
pip install --upgrade pip
pip install -r requirements.txt
```

### 3. 数据库连接失败
检查 `.env` 文件配置：
```env
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=your_password
MYSQL_DATABASE=pig_farm
```
